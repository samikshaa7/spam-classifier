# 📧 Spam Message Classifier

## 🔍 Overview

This project classifies messages as Spam or Not Spam using Machine Learning and Natural Language Processing.

## 🛠 Technologies Used

* Python
* Scikit-learn
* TF-IDF Vectorization
* Naive Bayes
* Streamlit

## ⚙️ Features

* User can input any message
* Real-time prediction
* Displays spam/not spam with confidence

## 🧠 How it Works

1. Text is converted into numerical form using TF-IDF
2. Naive Bayes model predicts whether the message is spam or not
3. Streamlit displays the result in a user-friendly interface

## ▶️ How to Run

1. Train model:
   python text_model.py

2. Run app:
   py -m streamlit run app.py

## 📊 Result

Achieved ~95% accuracy on dataset
